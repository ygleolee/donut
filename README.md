Animates a donut using ascii characters in the terminal haha. Users can control viewer location, lighting, rotation, and others!

## Requirements
- c++23
- xmake
- termios
- terminal supports ANSI escape codes

## Installation
- clone the repo
- `xmake install`
- done!
